In C-programming, control structure helps to manage the flow of a program. Control structure includes:

  • if statement
  • if-else statement
  • if-else-if statement
  • nested if statement
  • switch case


if statement

In C-language, the if statement is used for conditional execution. It means the block of code is execute if the given condition is true. Otherwise it does not execute.

syntax :-

if( condition )
{

//code to be executed

}


if-else statement

It contains a alternate code for execution if the given condition is false. The alternate code is written after the if statement by using else(keyword).

syntax :-

if( condition )
{

//code to be executed

}
else{

//alternate code to be executed

}


if-else-if statement

It contains a alternate code for execution if the first condition is false but the second condition is true. The alternate code is written in the second if statement followed by the else( keyword )

syntax :-

if( condition )
{

//code to be executed

}
else
if( condition )
{

//alternate code to be executed

}


nested if statement

It contains if statement inside another if statememt. The working principle is that first the condition of first( outer ) if condition is checked, if it is true then the condition of second( inner ) if condition is checked, if it is also true then the code is execute otherwise the code does not execute.

syntax :-

if( condition-1 )
{

if( condition-2 )
{
//code to be executed
}

}


switch case

switch case is used to replace multiple if statements. Instead 9f writing multiple if and conditions we simply use switch case to achieve our need in a shorthand way. switch keyword is used for accessing switch case.

syntax :-

switch ( variable )
{

case value1 : statement1;
statement2;
break;

case value2 : statement1;
statement2;
break;

case value3 : statement1;
statement2;
break;

default:statement1;
statement2;
break;

}

Variable :- It is the variable which value is checked by the computer.

case :- It is a keyword for condition checking.

values :- It is the values which is compared with the variable's value.

default :- If any of provided values does not match, the statements associated with default is executed.

break :- It is a keyword used to stop the further process( in that particular aspect ) after achieving our need.
